================================================================================
NOTES ON KEY FORMAT DIFFERENCES BETWEEN PLATFORMS
================================================================================

1. CATALYST 9300 (IOS-XE 17.x):
   - Uses "Switch uptime is" (not "Router uptime is")
   - Uses "(X86) processor" designation
   - Has Technology Package License table with Smart License types
   - System image file typically "flash:packages.conf" in INSTALL mode
   - Has "Switch Ports Model" table at the end
   - Includes AIR License Level lines
   - Uses Bengaluru/Cupertino/Dublin/Amsterdam codenames

2. ISR 4300/4400 (IOS-XE 16.x/17.x):
   - Uses "Router uptime is"
   - Uses "(1RU)" processor designation
   - Has Technology Package License table (appxk9, uck9, securityk9, ipbase)
   - System image in bootflash: (e.g., "bootflash:isr4300-universalk9.16.09.06.SPA.bin")
   - Uses Fuji/Gibraltar codenames for 16.x
   - Memory reported as xxxxK/6147K format

3. ASR 1001-X (IOS-XE 16.x/17.x):
   - Uses "Router uptime is" OR custom hostname + "uptime is"
   - Uses "(1NG) processor (revision 1NG)" designation
   - Has License Type/Level/Throughput lines (not Technology Package table)
   - System image in bootflash: (e.g., "bootflash:asr1001x-universalk9.16.09.03.SPA.bin")
   - Includes "The current throughput level is XXXX kbps" line
   - Has Smart Licensing Status line
   - Lists eUSB flash and WebUI ODM Files storage

4. CATALYST 3850 (IOS-XE 16.x):
   - Uses "Switch uptime is"
   - Has "c38xx Stack" in show inventory
   - Switch Ports Model table at the end of show version
   - System image file typically "flash:packages.conf"

5. CATALYST 2960/2960X (IOS 15.x):
   - Uses classic IOS format (no IOS-XE dual version lines)
   - Has BOOTLDR line with boot loader version
   - Lists MAC address and motherboard details
   - Has "Switch Ports Model" table at end
   - Configuration register typically 0xF or 0x102

6. show interfaces FORMAT DIFFERENCES:
   - Switch ports show "(connected)" after "line protocol is up"
   - Routers show just "line protocol is up" without "(connected)"
   - Switch ports may show "0 abort" in input errors line
   - Some outputs show "0 pause input"/"0 pause output"
   - BW units: some show "Kbit" others "Kbit/sec"
   - Input queue format varies: "0/75/0/0" vs "0/75/0"

7. show inventory FORMAT:
   - C9300: Uses "c93xx Stack" as top-level NAME
   - ISR 4331: Uses "Chassis" as top-level NAME
   - C3850: Uses "c38xx Stack" as top-level NAME
   - C2960X: Uses "1" as top-level NAME (switch number)
   - PID field width varies by device; trailing spaces pad to fixed width
   - VID/SN may be empty for modules without physical serial numbers
